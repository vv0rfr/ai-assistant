# AI智能助手

一个基于Flask的AI聊天助手，支持日常聊天、天气查询、穿搭建议、文档问答等功能。

## 功能特点

- 🤖 **智能对话** - 基于Mimo AI的自然语言对话
- 🌤 **天气查询** - 接入wttr.in免费天气API，提供实时天气信息
- 👗 **穿搭建议** - 根据天气自动推荐穿衣搭配
- 🧴 **护肤建议** - 根据温湿度提供护肤建议
- 🔐 **用户认证** - 手机号+验证码注册登录（开发模式验证码在终端显示）
- 💬 **历史记录** - 对话历史保存，支持搜索和管理（30天自动清理）
- 📄 **文档问答** - 上传PDF/Word/TXT，AI自动总结摘要并回答问题
- 🎨 **毛玻璃UI** - 深蓝紫渐变背景，磨砂玻璃效果，Linear+Notion+ChatGPT混合质感
- 📱 **响应式设计** - 支持桌面端和移动端

## 技术栈

- **后端**: Flask + SQLAlchemy + Flask-Login
- **数据库**: SQLite
- **天气API**: wttr.in（免费，无需API Key）
- **AI API**: Mimo AI（小米，OpenAI兼容格式）
- **前端**: HTML + CSS + JavaScript

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量（可选）

复制 `.env.example` 为 `.env`，按需配置：

- `SECRET_KEY` - Flask密钥（任意字符串）
- `MIMO_API_KEY` - Mimo AI API密钥（代码中有默认值）
- 天气API无需配置，使用免费的wttr.in

### 3. 运行应用

```bash
python app.py
```

访问 http://localhost:5000 即可使用。

### 4. 手机访问（同一网络）

1. 手机开热点，电脑连接热点
2. 查看电脑终端中显示的IP地址（如 `10.x.x.x`）
3. 手机浏览器访问 `http://<电脑IP>:5000`

> 注意：手机和电脑必须在同一网络下。电脑关机后手机无法访问。

## 短信验证码

开发模式下验证码在**电脑终端**显示，无需配置短信服务。终端输出格式：

```
==================================================
[开发模式] 短信验证码
手机号: 138xxxx
验证码: 123456
有效期: 5分钟
==================================================
```

## 天气查询

使用 wttr.in 免费天气API，无需API Key。支持：
- 直接查询：输入"北京天气"
- 省略格式：输入"焦作的呢"（基于上下文）
- 城市列表：内置约60个中国主要城市

## 启动脚本

- `start.bat` - 启动应用（命令行窗口）
- `open-assistant.bat` - 启动应用并打开浏览器
- `启动助手.vbs` - 静默启动（无命令行窗口）

## 常见问题

### Q: 验证码收不到？

A: 开发模式下验证码在电脑终端显示，不是手机收到。请查看运行 `python app.py` 的终端窗口。

### Q: 天气查询失败？

A: 确认输入的城市名在支持列表内（内置约60个城市）。wttr.in偶尔可能响应较慢。

### Q: 手机无法访问？

A: 确保手机和电脑在同一网络下。如果防火墙阻止，临时关闭防火墙测试。Windows自带防火墙和第三方安全软件（如火绒）都可能拦截。

### Q: 数据库在哪里？

A: SQLite数据库文件自动创建在 `instance/` 目录下。
